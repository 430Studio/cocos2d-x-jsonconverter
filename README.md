cocos2d-x-jsonconverter
=====================

Intro
---------------------
This is a simple JSON parser for cocos2d-x, based on cJSON.

It converts JSON objects to CCDictionary objects, and vice versa.


Datatype Mapping
---------------------

object : CCDictionary

array : CCArray

string : CCString

number : CCNumber

true/false : CCBool

null : CCNull
